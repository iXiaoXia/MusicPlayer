## 功能介绍
1. 音频播放控制：播放/暂停/上一首/下一首，支持MP3等主流格式
2. 播放模式：顺序播放、单曲循环、随机播放三种模式
3. 文件管理：支持文件夹选择，自动扫描并加载音频文件
4. 用户界面：自定义UI界面，支持列表展开/收起动画效果
5. 播放进度：可拖拽进度条控制播放位置，实时显示播放状态
